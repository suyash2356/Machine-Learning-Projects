## 🔍 What I Understood (Beyond Theory)

> 💡 *This section reflects real insights gained from projects.*

- **Algorithms are simple — data is hard**  
  → Clean data + good features matter more than complex models  

- **Math is not optional**  
  → Understanding equations makes debugging much easier  

- **Default parameters are not enough**  
  → Performance improves only when you experiment & tune  

- **Visualization = Understanding**  
  → Seeing decision boundaries & clusters builds intuition  

- **Every algorithm has assumptions**  
  → Performance depends on data distribution
